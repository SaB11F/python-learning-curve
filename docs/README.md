# Python Learning Docs

Study notes for my Python journey. Theory comes from the
[W3Schools Python course](https://www.w3schools.com/python/); the practical
code lives in [`../practical/`](../practical/).

## Convention

- One numbered doc per **major topic** (a major title in the W3Schools menu).
- Files are named `NNN-topic-name.md`, numbered in the order I learn them —
  like ADRs (`001`, `002`, `003`, …).
- Numbers never change once assigned, so the history stays stable.
- Each doc follows the same template (see below).

## Doc template

```
# NNN — Topic Name

- Source: <W3Schools link>
- Date: YYYY-MM-DD

## Key ideas        # the main points, in my own words
## Examples         # small code snippets, ideally ones I ran
## Gotchas          # things that surprised me or tripped me up
## Practice         # link to the file in ../practical/ where I tried it
## Takeaways        # the 1-3 things I want to remember
```
